# 校园外卖守护大作战 - 独立版使用说明

## 📖 概述

独立版是一个无需服务器、无需网络的纯客户端版本，学生只需双击HTML文件即可在浏览器中直接体验所有课程内容。

## 🎯 独立版特点

- ✅ **无需安装**：不需要安装Node.js或任何服务器软件
- ✅ **无需网络**：完全离线运行，不依赖网络连接
- ✅ **即开即用**：双击HTML文件即可在浏览器中打开
- ✅ **全部解锁**：所有任务默认解锁，学生可自由体验
- ✅ **数据本地化**：学生的学习进度自动保存在浏览器本地存储中

## 📦 文件列表

独立版包含以下文件：

```
第一课-我的专属外卖账号（独立版）.html
第三课-守护外卖账号（独立版）.html
独立版使用说明.md（本文件）
```

## 💻 系统要求

- **操作系统**：Windows、macOS、Linux均可
- **浏览器**：推荐使用以下任一现代浏览器
  - Chrome 90+ / Edge 90+
  - Firefox 88+
  - Safari 14+
  - 其他支持HTML5的浏览器

## 🚀 使用方法

### 方式一：教师发送给学生

1. 将HTML文件通过邮件、文件共享或U盘发送给学生
2. 学生将文件保存到电脑任意位置（建议保存到桌面）
3. 双击HTML文件，系统将自动在默认浏览器中打开
4. 学生即可开始体验课程内容

### 方式二：集中部署到教室电脑

1. 将HTML文件复制到教室所有电脑的共同位置（如：D:\课程资料\）
2. 在桌面创建快捷方式，方便学生快速访问
3. 学生双击快捷方式或文件即可开始学习

## 📚 课程内容说明

### 第一课：我的专属外卖账号

**学习目标**：
- 学会注册账号和填写个人信息
- 体验完整的外卖点餐流程
- 了解积分系统和用户信誉机制

**包含任务**：
- 任务一：注册账号（选择头像、填写信息、验证码）
- 任务二：点餐体验（选择餐厅、添加菜品、确认订单、支付、配送）
- 任务三：积分权益（查看积分、模拟行为、兑换奖励）
- 任务四：自我评价（学习效果自评）

### 第三课：守护外卖账号

**学习目标**：
- 理解密码安全的重要性
- 学会识别网络诈骗和钓鱼链接
- 掌握账号安全防护措施

**包含任务**：
- 任务一：密码安全（体验弱密码风险、学习设置强密码）
- 任务二：识别陷阱（识别诈骗信息、完成防骗知识测试）
- 任务三：安全中心（启用多重安全防护措施）
- 任务四：自我评价（安全意识自评）

## 💾 数据保存说明

### 数据存储位置
学生的学习进度（注册信息、积分、订单历史、评价等）会自动保存在浏览器的本地存储（localStorage）中。

### 数据特点
- **自动保存**：每次操作后自动保存，无需手动保存
- **持久保存**：关闭浏览器后数据不会丢失
- **本地隔离**：每个浏览器独立保存，不同浏览器的数据互不影响
- **文件绑定**：同一个HTML文件在同一浏览器中共享数据

### 数据清除
如果需要重新开始体验（清除所有进度）：

**Chrome/Edge浏览器**：
1. 按F12打开开发者工具
2. 切换到"Application"（应用程序）标签
3. 左侧找到"Local Storage" → 选择当前网址
4. 右键点击相应的键值 → 选择"Delete"（删除）
5. 刷新页面即可重新开始

**Firefox浏览器**：
1. 按F12打开开发者工具
2. 切换到"存储"标签
3. 左侧找到"本地存储" → 选择当前网址
4. 右键点击相应的项 → 选择"删除全部"
5. 刷新页面即可重新开始

## ⚠️ 注意事项

1. **浏览器兼容性**
   - 请使用现代浏览器打开，过旧的浏览器可能无法正常显示
   - 推荐使用Chrome、Edge、Firefox等主流浏览器

2. **文件路径**
   - 文件路径中不要包含特殊字符或中文（可能导致部分浏览器无法打开）
   - 建议将文件保存在简单的路径下，如桌面或D:\Lessons\

3. **弹窗拦截**
   - 部分浏览器可能拦截弹窗提示，请允许网页显示弹窗
   - 如果看不到提示信息，请检查浏览器地址栏右侧的弹窗拦截图标

4. **数据隔离**
   - 不同浏览器的数据是独立的（Chrome和Firefox的数据不互通）
   - 同一文件在不同位置的数据也是独立的

5. **隐私模式**
   - 在浏览器的隐私/无痕模式下，数据可能无法保存
   - 建议使用普通模式打开文件

## ❓ 常见问题

### Q1: 双击文件后没有反应？
**A**: 检查文件关联，确保HTML文件关联到浏览器。可以右键点击文件 → "打开方式" → 选择浏览器。

### Q2: 页面显示不正常？
**A**: 可能是浏览器版本过旧，请更新浏览器到最新版本，或尝试使用Chrome/Edge浏览器。

### Q3: 刷新页面后数据丢失？
**A**: 检查是否使用了浏览器的隐私/无痕模式。请使用普通模式打开文件。

### Q4: 验证码显示"请点击下方按钮获取"？
**A**: 这是正常的，需要先填写手机号，然后点击"获取验证码"按钮，系统会生成验证码。

### Q5: 如何查看学生的学习进度？
**A**: 独立版不支持教师端查看。如需查看学生进度，请使用服务器版，或让学生截图自评结果。

### Q6: 可以修改文件内容吗？
**A**: 可以，HTML文件是纯文本格式，可以用文本编辑器打开修改，但建议仅由有技术背景的教师进行修改。
