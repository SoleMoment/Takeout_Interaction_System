# 校园外卖守护大作战 - 教学系统部署指南（苏科版小学信息技术第四单元第一课和第三课）

## 📦 系统简介

这是一个基于Web的互动教学系统，包含两课内容：
- **第一课**：我的专属外卖账号（注册、点餐、积分系统）
- **第三课**：守护外卖账号（密码安全、防诈骗、安全设置）

## 🚀 快速部署（教师机）

### 方法一：一键启动（推荐）

#### Windows系统
1. 双击运行 `启动服务器.bat`
2. 看到"服务器启动成功"提示
3. 记下显示的IP地址（如：192.168.1.100）

#### Mac/Linux系统
1. 打开终端，进入项目目录
2. 运行：`bash 启动服务器.sh`
3. 记下显示的IP地址

### 方法二：手动启动

```bash
# 1. 安装依赖（首次运行）
npm install

# 2. 启动服务器
npm start
```

## 🌐 访问地址

### 教师机查看本机IP
- **Windows**: 打开命令提示符，输入 `ipconfig`，查看"IPv4地址"
- **Mac/Linux**: 打开终端，输入 `ifconfig` 或 `ip addr`

### 访问方式

假设教师机IP为 `192.168.1.100`，端口为 `3000`：

| 角色 | 访问地址 | 说明 |
|------|---------|------|
| 导航页 | http://192.168.1.100:3000 | 所有链接入口 |
| 学生-第一课 | http://192.168.1.100:3000/lesson1/student | 学生端操作界面 |
| 教师-第一课 | http://192.168.1.100:3000/lesson1/teacher | 教师控制台 |
| 学生-第三课 | http://192.168.1.100:3000/lesson3/student | 学生端操作界面 |
| 教师-第三课 | http://192.168.1.100:3000/lesson3/teacher | 教师控制台 |

## 📱 使用流程

### 课前准备

1. **教师机（服务器）**
   - 启动服务器程序
   - 打开浏览器访问教师控制台
   - 投影到大屏或一体机

2. **学生机**
   - 打开浏览器
   - 输入教师机IP地址
   - 选择对应课程进入学习

### 上课流程

1. **导入阶段**：教师介绍课程内容
2. **体验阶段**：学生在学生端完成任务
3. **控制阶段**：教师根据进度解锁任务
4. **监控阶段**：教师大屏实时查看学生数据
5. **评价阶段**：学生完成自我评价
6. **总结阶段**：教师导出数据分析

## 🔧 技术要求

### 服务器端（教师机）
- **操作系统**: Windows 7+, macOS 10.12+, Linux
- **Node.js**: 14.0 或更高版本
- **内存**: 至少 2GB RAM
- **网络**: 连接到局域网

### 客户端（学生机/教师机浏览器）
- **浏览器**: Chrome 90+, Edge 90+, Firefox 88+
- **网络**: 与教师机在同一局域网
- **分辨率**: 建议 1366x768 或更高

## 🛠️ 故障排查

### 问题1：学生机无法访问
**解决方案**：
1. 检查教师机防火墙是否开放端口3000
2. 确认教师机和学生机在同一局域网
3. 尝试 ping 教师机IP，确认网络连通

### 问题2：数据不同步
**解决方案**：
1. 检查浏览器控制台是否有错误
2. 刷新页面重新连接
3. 确认WebSocket连接正常（控制台显示"✅ WebSocket连接成功"）

### 问题3：服务器无法启动
**解决方案**：
1. 确认Node.js已正确安装：`node --version`
2. 确认依赖已安装：`npm install`
3. 检查端口3000是否被占用：
   - Windows: `netstat -ano | findstr :3000`
   - Mac/Linux: `lsof -i :3000`

### 问题4：防火墙设置

**Windows防火墙**：
1. 控制面板 → Windows Defender 防火墙 → 高级设置
2. 入站规则 → 新建规则
3. 端口 → TCP → 特定本地端口 → 3000
4. 允许连接

**Mac防火墙**：
1. 系统偏好设置 → 安全性与隐私 → 防火墙
2. 防火墙选项 → 添加应用程序 → Node

## 📊 数据导出

教师控制台提供CSV格式数据导出功能：
- 点击"导出数据"按钮
- 自动下载当前课程所有学生数据
- 包含：姓名、积分、评价、完成情况等

## 🔐 安全说明

- 系统仅在局域网内使用，不对外网开放
- 学生数据存储在浏览器本地（LocalStorage）
- 服务器不持久化存储学生数据
- 建议每节课前清空浏览器缓存重新开始
